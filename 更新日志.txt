移动端适配更新日志
====================

版本：v2.0 - 移动端全功能适配
更新日期：2025年

✅ 主要更新：

1. 设备自动检测
   - 后端通过User-Agent自动识别PC和移动设备
   - 支持iPhone、iPad、Android、Windows Phone等设备

2. 移动端UI优化
   - 控制面板：缩小padding和字体，更紧凑
   - 景区标记：圆点从20px缩小到16px
   - 名称标签：字体从11px缩小到9px，最大宽度80px
   - 隐藏统计卡片：移动端不显示右下角统计信息
   - 新增全屏按钮：替代统计卡片，位于右下角

3. 移动端交互优化
   - 点击景区：直接调用小红书App（xhsdiscover://search/result）
   - 无网页跳转：避免移动端浏览器的页面切换
   - 全屏功能：支持进入/退出全屏模式

4. 技术特性
   - CSS媒体查询：@media (max-width: 640px)
   - 跨浏览器全屏API：支持webkit、moz、ms前缀
   - 响应式设计：自动适配不同屏幕尺寸

✅ PC端功能保持不变：
   - 景区标记鼠标悬浮显示名称
   - 点击在新窗口打开小红书网页版
   - 显示统计卡片（右下角）
   - 保持原有控制面板样式

📝 使用说明：

PC端：
1. 浏览器访问 http://127.0.0.1:5000
2. 选择底图和景区等级
3. 鼠标悬浮在圆点上查看景区名称
4. 点击圆点在新标签页打开小红书搜索

移动端：
1. 手机浏览器访问 http://[电脑IP]:5000
2. 选择底图和景区等级
3. 景区标记直接显示名称
4. 点击标记打开小红书App搜索
5. 点击右下角按钮进入全屏模式

⚠️ 注意事项：

1. 移动端需要安装小红书App才能跳转
2. 某些浏览器可能会拦截URL Scheme
3. iOS可能需要用户授权允许打开App
4. 全屏功能需要浏览器支持Fullscreen API
5. 建议使用现代移动浏览器（Chrome、Safari等）
